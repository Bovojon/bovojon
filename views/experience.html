Xtern BootCamp, powered by TechPoint, is an intensive 10-week training program focused on software development, during which I was trained to build single-page web applications using the MEAN (MongoDB, Express.js, Angular, and Node.js) software stack. When comparing the amount of time it took to build the front end of a full-stack note-taking web application with plain vanilla JavaScript at first, and then jQuery and later AngularJS, it was fascinating to see how much of the work AngularJS managed behind the scenes, and how much time and lines of code were saved! It was then that I realized how powerful the model–view–controller (MVC) design pattern really was.

During the BootCamp, we also had two 8-hour days of rigorous Scrum software development training. Scrum is an incremental Agile software development framework for managing product development. Towards the end, we applied what we had learnt by putting our knowledge of Scrum to the test and building layouts of towns using legos while under workplace conditions with deadlines to meet. As a team we created our sprint backlogs and met our priorities in our 'daily' scrum meetings. While formulating our sprint backlogs I realized how important and effective version control systems (such as git) were in distributing work and then merging each team member's work to form the final product at the end.

While building full-stack applications, I also worked with RESTful APIs, JSON objects, Ruby on Rails and PostgreSQL. All of my projects that I built during this BootCamp can be found on my github page.
